# Changelog v5.3.45

## Date: 18 Janvier 2026

## Améliorations

### Génération d'images améliorée
- **Description physique détaillée**: Les images générées tiennent maintenant parfaitement compte de:
  - Couleur et longueur des cheveux
  - Couleur des yeux
  - Taille (petite, moyenne, grande)
  - Morphologie corporelle
  - Taille de poitrine (femmes)
  - Couleur de peau
- **Morphologie "ronde"**: Ajoute maintenant un léger ventre (`small soft belly, slight tummy curve`)
- **Morphologie "très ronde"**: Ajoute un ventre plus important (`PROMINENT ROUND BELLY, big soft tummy`)
- **Termes ignorés**: "voluptueuse" et "généreuse" ne génèrent plus de corps gros - traités comme corps normal avec formes
- **Traduction des couleurs**: Meilleure conversion français -> anglais pour cheveux, yeux, peau

### Conversations
- **Affichage corrigé**: Les conversations anciennes (formats legacy) sont maintenant retrouvées
- **Recherche élargie**: Support des formats `conv_anonymous_`, `conv_device_`, `conversation_`, `conv_default_`
- **Personnage placeholder**: Si un personnage n'est pas trouvé, un placeholder est créé pour afficher la conversation
- **Robustesse améliorée**: Plus de conversations "perdues" à cause de changements d'ID utilisateur

### Import d'images
- **Sans redimensionnement**: L'image complète est gardée, pas de recadrage 1:1 forcé
- **Qualité élevée**: Qualité augmentée à 0.9 (au lieu de 0.8)
- **Auto-génération description**: Quand une image est importée, la description physique détaillée est auto-générée
- La description inclut: âge, taille, morphologie, poitrine/pénis, cheveux, yeux, peau

## Fichiers modifiés
- `src/services/ImageGenerationService.js` - Nouvelle fonction `generateCharacterImage()` avec description détaillée
- `src/services/StorageService.js` - `getAllConversations()` recherche tous les formats
- `src/screens/ChatsScreen.js` - Placeholder pour personnages non trouvés
- `src/screens/CreateCharacterScreen.js` - Import sans crop + auto-génération description
- `app.json` - Version 5.3.45, versionCode 99
- `package.json` - Version 5.3.45

- La fonction `autoGenerateDescription()` crée automatiquement la description physique
- Les termes "voluptuous" et "generous" sont évités dans les prompts d'image
