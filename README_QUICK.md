# Roleplay Chat App

Une application mobile Android de roleplay conversationnel avec 200 personnages uniques.

## Installation rapide

```bash
npm install
npm start
```

Scannez le QR code avec Expo Go sur Android.

## Configuration

1. Obtenez des clés API gratuites sur console.groq.com
2. Ajoutez-les dans l'onglet Paramètres de l'app
3. Commencez à discuter !

Voir README.md pour la documentation complète.
